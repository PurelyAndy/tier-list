Press Ctrl+E to export your tier list as an image, or Ctrl+S to save it as a JSON file.

Empty slots will be omitted when exporting the tier list, and long rows will be wrapped to the next line.

You can scroll horizontally if a tier has too many items to fit.

Drag and drop a saved JSON file to load a tier list.

Images can be pasted or dragged into the slots.

Press F1 at any time to show this help message again.
